# SpendWise - Personal Finance Tracker

SpendWise is a web application that helps users manage their personal finances by tracking income and expenses, viewing spending trends, and monitoring their financial health.

## Features

- User Authentication (Supabase)
- Add Income
- Add Expense
- Dashboard with financial summary
- Expense Analytics
- Transaction History
- Responsive UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase

## Installation

Clone the repository:

```bash
git clone https://github.com/Chandru13k/Spendwise1.git
```

Navigate to the project:

```bash
cd Spendwise1
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_publishable_key
```

Run the application:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
